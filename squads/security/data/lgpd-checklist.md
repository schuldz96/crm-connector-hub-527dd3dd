# LGPD — Checklist de Conformidade

## Princípios (Art. 6)
- [ ] **Finalidade** — Propósitos legítimos e informados ao titular
- [ ] **Adequação** — Tratamento compatível com as finalidades
- [ ] **Necessidade** — Coleta mínima necessária
- [ ] **Livre acesso** — Consulta fácil sobre o tratamento
- [ ] **Qualidade** — Dados exatos e atualizados
- [ ] **Transparência** — Informações claras sobre o tratamento
- [ ] **Segurança** — Medidas técnicas de proteção
- [ ] **Prevenção** — Medidas para evitar danos
- [ ] **Não discriminação** — Tratamento não discriminatório
- [ ] **Responsabilização** — Demonstrar conformidade

## Bases Legais (Art. 7)
- [ ] Cada dado pessoal tem base legal definida
- [ ] Consentimento é específico, livre e informado
- [ ] Legítimo interesse tem LIA documentado

## Direitos do Titular (Arts. 17-22)
- [ ] Confirmação de existência de tratamento
- [ ] Acesso aos dados
- [ ] Correção de dados incompletos/incorretos
- [ ] Anonimização/bloqueio/eliminação de dados desnecessários
- [ ] Portabilidade
- [ ] Eliminação (revogação de consentimento)
- [ ] Informação sobre compartilhamento
- [ ] Revogação do consentimento

## Segurança e Boas Práticas (Arts. 46-50)
- [ ] Medidas técnicas e administrativas de proteção
- [ ] Comunicação de incidentes à ANPD (Art. 48)
- [ ] Comunicação de incidentes aos titulares
- [ ] DPO/Encarregado designado (Art. 41)
- [ ] Registro de atividades de tratamento (Art. 37)
