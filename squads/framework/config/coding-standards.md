# Coding Standards — Squad Framework

## Padrões de Avaliação

### Scoring
- Escala universal: 0-100
- Red flags: -10 pontos cada
- Green flags: +5 pontos cada (máximo +15)
- Dimensões de avaliação: escala 1-5, multiplicada pelo peso

### Classificações
- 90-100: Master
- 75-89: Proficient
- 60-74: Developing
- 40-59: Beginner
- 0-39: Non-adherent

### Output
- Relatórios em Markdown
- Tabelas para scores comparativos
- Listas para action items
- Emojis apenas para status indicators (✅ ❌ ⚠️)

### Linguagem
- Feedback em português brasileiro
- Termos técnicos das metodologias em inglês
- Tom construtivo e acionável
